# Final Word Document Formatting

## ✅ **Changes Made by `format_final_word_documents.py`**

This script applies your requested formatting changes to the final Word documents.

---

## 📝 **What Gets Changed**

### **1. Make Titles/Subtitles Bold**
**BEFORE:**
```
Category: 1.1 Personal Information
Page: 2
Confidence: 0.98
Content: Salary information...
Reason: Contains salary details
```

**AFTER:**
```
**Category:** 1.1 Personal Information
**Page:** 2
**Confidence:** 0.98
**Content:** Salary information...
**Reason:** Contains salary details
```

**Also makes bold:**
- ✅ Total Names Extracted:
- ✅ Total Classifications:
- ✅ Extracted Names
- ✅ Summary Statistics
- ✅ Classification Breakdown

---

### **2. Remove "Sensitive Content Classifications"**
**BEFORE:**
```
Segment: segment_1
Pages: 1-3

Sensitive Content Classifications    ← REMOVED
Total Classifications: 2

Classification 1
...
```

**AFTER:**
```
Segment: segment_1
Pages: 1-3

Total Classifications: 2

Classification 1
...
```

---

### **3. Remove "Report generated by Azure DI Output Parser"**
**BEFORE:**
```
Classification Breakdown
1.1 Personal Information: 1
2.1 CV or Resume Content: 1

Report generated by Azure DI Output Parser    ← REMOVED
```

**AFTER:**
```
Classification Breakdown
1.1 Personal Information: 1
2.1 CV or Resume Content: 1
```

---

### **4. Remove Duplicate "Total Segments" and "Total Pages"**
**BEFORE:**
```

Total Pages: 5                    ← REMOVED (duplicate)
Total Segments: 2                 ← REMOVED (duplicate)

Total Names Extracted: 3
Total Classifications: 4
Total Segments: 2                 ← KEPT (in summary)
Total Pages: 5                    ← KEPT (in summary)
```

**AFTER:**
```

Summary Statistics
Total Names Extracted: 3
Total Classifications: 4
Total Segments: 2
Total Pages: 5
```

## 🚀 **How to Use**

### **Run the Script**
```python
python format_final_word_documents.py
```

### **What It Does**
- ✅ Processes all `.docx` files in `final_word_reports/` folder
- ✅ Makes requested changes to each file
- ✅ Saves changes back to the same files (overwrites)
- ✅ Uses temp file workaround for Databricks Volumes
